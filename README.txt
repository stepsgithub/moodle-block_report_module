安裝步驟：
1. 請將report_module資料夾放置在moodle資料夾位置的blocks之內
2. 點選 網站管理 > 通知  =>區塊安裝結果確認
3. 如安裝正常請將『報表模組』區塊新增至合適的位置(建議可放置在系統管理之下)

使用說明：
1. 一開始請將'完成'的課程做設定 (在『課程標準設定』使用設定)
2. 須在課程設定完成之後才可以正確的使用功能
3. 如須知道通過率須至『課程通過率計算』使用
4. 必須具有教師權限才可以使用報表模組的相關功能

(註：由於每日會自動將學習報表傳送至FTP，所以請在block_report_module.php中的FTP_HOST(FTP主機)、FTP_USER(FTP帳號)、FTP_PASSWORD(FTP密碼)取代成符合環境的值)
